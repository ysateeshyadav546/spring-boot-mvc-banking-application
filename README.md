# 💳 Spring Boot MVC Banking Application

## 📌 Overview
This is a full-stack Banking Application developed using Spring Boot following MVC architecture. It supports account management, fund transfer, and transaction tracking.

---

## 🚀 Features
- User Login & Registration
- Account Management
- Deposit & Withdrawal
- Fund Transfer
- Transaction History

---

## 🛠 Tech Stack
- Java
- Spring Boot
- JDBC
- MySQL (SQL)
- HTML, CSS, Bootstrap
